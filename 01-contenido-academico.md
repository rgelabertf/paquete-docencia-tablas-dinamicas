# Guía de Estudio — Tablas Dinámicas en Excel

## 1. ¿Qué son las Tablas Dinámicas?
Una tabla dinámica (PivotTable) es una herramienta de Excel que permite resumir, analizar y presentar grandes volúmenes de datos de forma interactiva, sin escribir fórmulas. Con solo arrastrar y soltar campos, puedes reorganizar el análisis en segundos.

## 2. Preparación de Datos
- Los datos deben tener encabezados en la fila 1
- Sin filas o columnas completamente vacías
- Sin celdas combinadas
- Convertir el rango en Tabla de Excel (Ctrl + T) para actualización automática

## 3. Cómo Crear una Tabla Dinámica
1. Selecciona cualquier celda dentro de tus datos
2. Ve a **Insertar → Tabla dinámica** (o Alt + N + V)
3. Elige "Nueva hoja de cálculo" y haz clic en Aceptar
4. Arrastra campos a las áreas: Filas, Columnas, Valores, Filtros
5. Personaliza los cálculos (suma, promedio, conteo, etc.)

## 4. Las 4 Áreas de una Tabla Dinámica
- **Filas:** Organizan datos de forma vertical (ej. nombres de productos)
- **Columnas:** Organizan datos de forma horizontal (ej. meses del año)
- **Valores:** Los datos numéricos que se resumen (ej. ventas, cantidades)
- **Filtros:** Permiten mostrar solo ciertos datos (ej. una región específica)

## 5. Personalización y Formato
- Cambiar diseño de informe (compacto, tabular, esquemático)
- Agregar segmentaciones para filtrar visualmente
- Insertar escalas de tiempo para filtrar por fechas
- Aplicar estilos y formato de números
- Mostrar valores como porcentaje del total

## 6. Gráficos Dinámicos
Los gráficos dinámicos se vinculan directamente a tu tabla dinámica. Al modificar la tabla, el gráfico se actualiza automáticamente. Ideales para crear dashboards visuales.

## 7. Errores Comunes
- ❌ No actualizar la tabla después de cambiar los datos de origen
- ❌ Tener datos en blanco o con formatos inconsistentes
- ❌ Usar celdas combinadas en los datos de origen
- ❌ No usar Tablas de Excel como origen de datos
- ✅ Solución: Convertir datos a Tabla (Ctrl+T) y usar "Actualizar" (clic derecho)

## Flashcards

| Pregunta | Respuesta |
|----------|-----------|
| ¿Qué es una tabla dinámica? | Herramienta de Excel que resume, analiza y presenta grandes volúmenes de datos de forma interactiva. |
| ¿Cuáles son las 4 áreas de una tabla dinámica? | Filas, Columnas, Valores y Filtros. |
