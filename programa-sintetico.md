## Competencia General

Aplica la funcionalidad de tablas dinámicas y gráficos dinámicos en Excel para el análisis, organización y presentación visual de conjuntos de datos provenientes de contextos académicos y cotidianos, mediante la preparación adecuada de datos, el uso de campos calculados, segmentaciones y dashboards, cultivando el pensamiento crítico, la toma de decisiones basada en datos y el trabajo colaborativo.

---

## Competencias Específicas

1. **Prepara** conjuntos de datos aplicando técnicas de limpieza, normalización y estructuración tabular, para garantizar la integridad y confiabilidad de la información antes de su análisis con tablas dinámicas.

2. **Construye** tablas dinámicas multidimencionales configurando las áreas de filas, columnas, valores y filtros, así como aplicando formatos condicionales y estilos de diseño, para la síntesis interactiva de grandes volúmenes de datos.

3. **Analiza** información mediante campos calculados, agrupaciones, ordenamientos avanzados y funciones de agregación (mostrar valor como), interpretando tendencias, patrones y anomalías para fundamentar conclusiones y propuestas.

4. **Diseña** dashboards interactivos integrando tablas dinámicas, gráficos dinámicos, segmentaciones de datos y escalas de tiempo, para comunicar visualmente hallazgos de forma clara, dinámica y profesional.

## Estrategias de Enseñanza-Aprendizaje

### Estrategias de Enseñanza (Docente)

1. **Exposición didáctica** con demostraciones en vivo del software Excel para introducir cada nuevo concepto y funcionalidad.
2. **Aprendizaje basado en problemas (ABP):** Planteamiento de casos reales (ventas, inventarios, calificaciones, encuestas) que requieran análisis mediante tablas dinámicas.
3. **Tutoriales guiados:** Ejercicios paso a paso con instrucciones detalladas para la construcción de tablas y dashboards.
4. **Estudio de casos:** Análisis de dashboards profesionales para identificar buenas prácticas y elementos de diseño efectivo.
5. **Acompañamiento personalizado** durante las sesiones prácticas para resolver dudas específicas.
6. **Uso de plataforma educativa** (Classroom / Teams) para distribución de materiales, entrega de tareas y retroalimentación.

### Estrategias de Aprendizaje (Estudiante)

1. **Ejercicios prácticos** con conjuntos de datos proporcionados y autogestionados.
2. **Portafolio de evidencias** que integre los productos generados en cada unidad.
3. **Proyecto integrador:** Construcción de un dashboard completo a partir de un conjunto de datos de su interés.
4. **Bitácora de aprendizaje** donde registre comandos, atajos y procedimientos aprendidos.
5. **Revisión entre pares (coevaluación)** de dashboards y tablas dinámicas.
6. **Autoevaluación** del logro de competencias al final de cada unidad mediante rúbricas.
